## ✅ NOTE SOURCE — EXTENSIONS NAVIGATEUR

Objectif :
- garder une base de recommandations par rôle
- éviter les dumps géants de Chrome Web Store
- ne faire remonter dans le cours que les extensions qui améliorent vraiment un workflow

## Catégories utiles

### 1. Focus / friction reduction

- ((uBlock Origin))
- ((Dark Reader))
- ((News Feed Eradicator))
- ((I don't care about cookies comme outil de confort / anti-popups, pas comme outil privacy))
- ((Unhook pour YouTube))

### 2. Capture / lecture / recherche

- ((SingleFile))
- ((Readwise Highlighter))
- ((Jiffy Reader))
- ((Google Dictionary))
- ((Project Naptha))
- ((Selection Search))

### 3. Email / Gmail

- ((Simplify Gmail))
- ((cloudHQ comme boîte à outils Gmail plutôt qu'une seule extension))
- ((Gmelius, Mixmax, etc. seulement selon usage pro réel))

### 4. Tabs / sessions / bookmarks

- ((Toby))
- ((BrainTool))
- ((TabXpert))
- ((Snooze Tabs / Snoozz selon usage))

### 5. YouTube / vidéo / apprentissage

- ((PocketTube))
- ((Video Speed Controller))
- ((Video Resumer))
- ((ReClipped / Snipo / YiNote selon prise de notes vidéo))
- ((FreshView / playlist helpers / watch later helpers comme secondaires))

### 6. Web clipping / notes / PKM

- ((Notion Web Clipper))
- ((Save to Notion))
- ((Obsidian Web comme accès rapide, non central))
- ((Eloquent / quick capture tools seulement si le workflow le justifie))

### 7. Dev / SEO / scraping

- ((SelectorsHub))
- ((Talend API Tester))
- ((Web Scraper))
- ((Lighthouse))
- ((Ahrefs SEO Toolbar))
- ((Link Redirect Trace))

### 8. Automation / text expansion / form filling

- ((Automa))
- ((Magical))
- ((FlyMSG / Text Blaze / autres si besoin d'autofill))
- ((Violentmonkey pour scripts utilisateurs))

## Recommandations à garder pour Winflowz

- ne recommander dans le cours que des extensions :
  - toujours maintenues
  - vraiment différenciantes
  - rattachées à un problème précis
- éviter les longues listes d'extensions quasi équivalentes
- préférer un cadre de décision par rôle :
  focus, capture, Gmail, tabs, YouTube, dev, automation

## 🔶 À TRIER

- à revoir seulement si un module spécifique l'exige :
  grosses listes YouTube, SEO, scraping, IA writing, CRM, multi-login, image downloaders
- ne pas réinjecter tels quels :
  export brut Chrome Web Store, doublons, extensions non vérifiées, outils marketing opportunistes
