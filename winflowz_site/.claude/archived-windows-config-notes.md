## Windows Auto-Login

Source references:
- `netplwiz`
- `control userpasswords2`
- `AutoLoginOK`
- Registre `Winlogon`

Key idea:
- autologin on Windows can remove friction on a personal machine

Important caveats:
- only acceptable on a trusted home PC
- bad idea on shared or exposed machines
- extra complications with Microsoft accounts and PIN-based setups
