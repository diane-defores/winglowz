Oui, il est possible de changer automatiquement entre le mode sombre et clair en fonction de l'heure du coucher du soleil sur Windows 11, mais pas avec les fonctionnalités natives du système d'exploitation. Pour ce faire, vous devez utiliser une application tierce appelée Auto Dark Mode[1][2][3].

## Fonctionnement d'Auto Dark Mode

Auto Dark Mode est une application gratuite qui permet de planifier le basculement entre les thèmes sombre et clair sur Windows 11. Voici comment l'utiliser :

1. Téléchargez et installez Auto Dark Mode depuis le Microsoft Store ou GitHub[3].

2. Lancez l'application à partir du menu Démarrer.

3. Dans les paramètres de l'application, vous pouvez choisir l'option "From sunset to sunrise" (Du coucher au lever du soleil)[4].

4. L'application vous demandera l'autorisation d'accéder à votre localisation pour déterminer les heures exactes de lever et de coucher du soleil dans votre région[4].

## Fonctionnalités supplémentaires

Auto Dark Mode offre également d'autres options de personnalisation :

- Définir des heures personnalisées pour le changement de thème[1][3].
- Changer automatiquement le fond d'écran, le curseur et la couleur d'accentuation[3].
- Configurer des exceptions pour certaines applications[1].
- Retarder le changement de thème si vous êtes en train de jouer à un jeu vidéo[1].

Cette application compense efficacement l'absence de cette fonctionnalité native dans Windows 11, offrant une solution pratique pour adapter automatiquement l'apparence de votre système en fonction de l'heure de la journée[2][3].
