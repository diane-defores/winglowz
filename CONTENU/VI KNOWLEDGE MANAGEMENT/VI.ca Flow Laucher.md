## Location
Choose portable location in case you switch computer and choose a folder easy to find (I.E at the root of your drive), so you’ll have to do all of these configs only once.
## Search bookmarks
To be able to search for bookmark in Vivaldi go to Parameters - Plugins - Bookmarks - Add
![|529](flow-launcher-bookmark-plugin.png)

You can find the path to your bookmarks by typing “vivaldi://about/” in your Vivaldi’s address bar , and look for the path to your user profile.

![|837](flow-launcher-vivaldi-path.png)

Enter it into Flow settings.

![|353](flow-launcher-settings-bookmarks.webp)


I can now search for my Vivaldi bookmarks inside Flow!
![](flow-launcher-bookmark-search.webp)
## Action keyword and Priority
You can have an action keyword to trigger the bookmark search. The default one is “b” as you can see on the screenshort before. If you don’t want any like me, just delete it or use ```*``` instead, the search will work from the get-go. You can choose a priority number for bookmarks to be shown at the start or at the end of results.
![|749](flow-launcher-action-keyword.png)
