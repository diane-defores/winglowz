# BUG-2026-05-16-011 - Media brightness buttons need Android system settings access

Status: fixed-pending-verify
Date: 2026-05-16
Area: Android keyboard / media action row / onboarding

## Report
The media action bar needs brightness controls, but Android requires the special `WRITE_SETTINGS` permission before an app can modify `Settings.System.SCREEN_BRIGHTNESS`.

## Fix
- Added `Bri-` and `Bri+` media action row buttons.
- Added native brightness adjustment through `Settings.System.SCREEN_BRIGHTNESS`.
- Added `WRITE_SETTINGS` manifest permission and a native bridge to open Android's write-settings permission screen.
- Added keyboard status field `systemSettingsWriteGranted`.
- Added onboarding step `LuminositĂŠ systĂ¨me`.
- If permission is missing, `Bri-`/`Bri+` opens `/settings?onboarding=brightness` instead of silently failing.
