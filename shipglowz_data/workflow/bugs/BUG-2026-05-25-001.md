## Expected Behavior

- The Android debug APK job compiles Kotlin sources and produces the CI debug APK artifact.

## Observed Behavior

- `KeyboardLayoutModels.kt` contained Kotlin syntax invalid for CI compilation: trailing commas inside a `when` expression and escaped quotes inside a regular string interpolation.
- `WinGlowzInputMethodService.kt` logged `.applied` on values already converted to `Boolean` by `reportFailure`.
- The parser errors cascaded into unresolved helper functions and blocked the APK job.
