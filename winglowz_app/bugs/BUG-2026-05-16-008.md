## Summary

- Title: Android keyboard Début moves to paragraph start instead of full text start
- Current status: `fixed-pending-verify`
- Severity: `medium`
- Owner: Diane
- Next command: `/sf-test --retest BUG-2026-05-16-008`

## Reproduction

- Preconditions:
  - Android real device.
  - WinGlowz keyboard active.
  - Multi-line or multi-paragraph text field with cursor away from the start.
- Steps:
  1. Open the Navigation panel or attached navigation row.
  2. Tap `Début`.

## Expected Behavior

- Cursor moves to the very beginning of the whole text.
- `Fin` should symmetrically move to the very end of the whole text.

## Observed Behavior

- `Début` moves only to the start of the current paragraph/line area.

## Evidence

- User confirmation in chat at 2026-05-16.
